# Бот для записи к специалисту (VK)

[![License](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)
[![Python](https://img.shields.io/badge/Python-3.10%2B-blue)](https://www.python.org/)
[![VK API](https://img.shields.io/badge/VK%20API-5.199-blue)](https://dev.vk.com/)

Бот для платформы ВКонтакте, который автоматизирует запись клиентов к специалистам. Управление услугами, мастерами, расписанием и бронированием через удобный интерфейс с кнопками.

## 📋 Функционал

### Для клиентов
- **Выбор услуги** из каталога (цена, длительность)
- **Выбор мастера** с описанием специализации
- **Интерактивный выбор даты** (только доступные дни)
- **Выбор свободного времени** из динамических слотов
- **Подтверждение записи** с отображением всех деталей
- **Просмотр активных записей** (`/my_bookings`)
- **Самостоятельная отмена записи** через кнопку в списке

### Для администратора
- Управление каталогом услуг (`services` таблица)
- Управление мастерами (`masters` таблица)
- Гибкое управление слотами (интеграция с календарем)

### Технические особенности
- 🔘 **Инлайн-клавиатуры** для всех действий
- 💾 **Постоянное хранение** данных в SQLite
- 🔄 **Callback-обработка** нажатий на кнопки (`MESSAGE_EVENT`)
- 🧠 **Конечный автомат** (FSM) для управления диалогом
- 📝 **Полное логирование** для отладки

## 🚀 Быстрый старт

### 1. Клонирование репозитория
```bash
git clone https://github.com/ваш-аккаунт/booking_bot.git
cd booking_bot


2. Настройка виртуального окружения:
python -m venv venv
source venv/bin/activate      # Linux/Mac
venv\Scripts\activate          # Windows


3. Установка зависимостей:
pip install -r requirements.txt


4. Настройка окружения
Создайте файл .env в корне проекта:
VK_TOKEN=ваш_токен_сообщества
VK_GROUP_ID=123456789
DATABASE_PATH=booking.db


Как получить токен:
Создайте сообщество ВКонтакте
Управление → Работа с API → Создать ключ
Дайте права: Сообщения сообщества и Управление сообществом
Скопируйте токен


5. Инициализация базы данных:
python -m storage.database   # Создаёт таблицы
python add_slots.py          # Добавляет тестовые слоты. Запускать как при первоначальном использовании бота, так и когда своюодные слоты заканчиваются.

## Управление расписанием

В демо-версии слоты создаются скриптом `add_slots.py`.
В реальном проекте используется:
- Генерация слотов на основе шаблона рабочего дня
- Админ-панель (Flask) для управления расписанием
- Возможность блокировки отдельных слотов вручную
- Интеграция с Google Calendar (опционально)


6. Запуск бота:
python bot.py


🎮 Команды бота:
Команда	Действие

/my_bookings	Показать мои записи
/help	Справка

🗄 Схема базы данных:
services     - Услуги (id, name, duration, price)
masters      - Мастера (id, name, description)
slots        - Слоты (id, master_id, service_id, start_time, end_time)
bookings     - Записи (id, user_id, slot_id, service_id, master_id, status)
